# Predictive Demand Modelling for Mobility-as-a-Service (MaaS)

## Overview
This research project investigates the use of machine learning to model and predict demand for Mobility-as-a-Service (MaaS) systems. It proposes a smart city framework aimed at improving transportation efficiency, accessibility, and sustainability, with a focus on transferability to rapidly urbanising African cities.

---

## Objectives
- Develop predictive models for multimodal transport demand  
- Analyse travel behaviour patterns in urban systems  
- Design a scalable MaaS framework for smart cities  
- Evaluate transferability to African urban contexts  
- Support data-driven transport planning  

---

## Research Focus
This work sits at the intersection of:
- Transportation Engineering  
- Travel Behaviour Analysis  
- Machine Learning  
- Smart Cities and Urban Systems  

Key areas include:
- Multimodal mobility systems  
- Demand forecasting  
- Mobility integration (public transport, micromobility, ride-sharing)  
- Accessibility and sustainability

## Methodology

### 1. Data Collection

### 2. Data Processing

### 3. Model Development
- Machine learning models (regression, time-series, neural networks)  
- Demand prediction for MaaS systems  

### 4. Framework Design
- Integration into a smart city MaaS framework  
- Scenario simulation  

### 5. Evaluation
- Model performance assessment  
- Transferability analysis for African cities
